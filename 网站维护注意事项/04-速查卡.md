# 维护速查卡

## 紧急排查三步

1. **网站打不开** → `ssh` 进服务器 → `nginx -t && nginx -s reload`
2. **功能异常** → 检查 `functions.php` 最近是否被改坏了 → 对比 `.bak` 备份
3. **搜索出问题** → 看 `02-搜索功能.md`

## 改代码流程

```
1. cp functions.php functions.php.bak.$(date +%Y%m%d_%H%M%S)
2. 改 /www/wwwroot/crrg_moebius_com_114/wp-content/themes/astra-child/functions.php
3. 刷新网页测试
4. 确认无误 → 删旧 bak（保留最近 2 个即可）
```

## 常用命令

```bash
# 进入主题目录
cd /www/wwwroot/crrg_moebius_com_114/wp-content/themes/astra-child/

# 查看最近修改
ls -lt *.php | head -5

# 对比备份
diff functions.php functions.php.bak.20260707_135654

# 恢复备份
cp functions.php.bak.20260707_135654 functions.php

# 清理旧备份（保留最近 2 个）
ls -t functions.php.bak.* | tail -n +3 | xargs rm -f
```
